# Mutual_Funds_Investment# 📊 Mutual Fund Investment Analysis Dashboard

## Overview

This project is an interactive **Power BI dashboard** designed to analyze and compare mutual funds based on key financial metrics such as returns, risk, AUM, expense ratio, and fund ratings. The goal is to help investors identify top-performing funds and make data-driven investment decisions.

---

## Problem Statement

Investors often struggle to compare hundreds of mutual funds across multiple performance indicators. This project simplifies the decision-making process by providing a centralized dashboard for fund analysis and ranking.

---

## Objectives

* Analyze mutual fund performance using historical returns
* Compare risk and return metrics across funds
* Rank top-performing funds using a scoring model
* Visualize AMC and category-wise fund performance
* Identify investment opportunities using data insights

---

## Dataset

The dataset contains mutual fund information such as:

* Fund Name
* AMC Name
* Category
* Assets Under Management (AUM)
* Expense Ratio
* 1-Year Return
* 3-Year Return
* 5-Year Return
* Risk Metrics
* Ratings

---

## Tools & Technologies Used

* **Microsoft Excel** — Data cleaning and preprocessing
* **Power BI** — Dashboard creation and visualization
* **DAX** — Calculated columns, measures, and scoring model
* **Data Analysis** — Ranking and performance comparison

## Dashboard Features

### 📌 Key KPIs

* Total Funds Analyzed
* Average Returns
* Average Expense Ratio
* Top Ranked Funds

### 📈 Visualizations

* Risk vs Return Analysis
* Top 30 Mutual Funds Ranking
* AMC-wise Performance Comparison
* Category-wise Distribution
* Return Trend Analysis

---

## Methodology

1. Collected and cleaned mutual fund dataset
2. Handled missing and inconsistent values
3. Created normalized scoring metrics for comparison
4. Ranked funds based on weighted performance indicators
5. Built interactive Power BI dashboard for insights

---

## Key Insights

* Certain funds consistently outperform peers across multiple return periods
* Lower expense ratio does not always indicate better performance
* Some AMCs dominate top-performing fund categories
* Risk-adjusted returns provide better investment insights than raw returns alone
